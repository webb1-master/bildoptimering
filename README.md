# bildoptimering
Bildoptimering och srcset

Målet med uppgiften är att ni ska få bättre förståelse för hur man kan använda sig av responsiv design. Utöver det introducerar vi även taggen <picture> som underlättar bildhantering inom responsiv design.

Utgå från html och css i detta repo.

- Optimera bilden med solnedgång. (Den ska finnas i tre format, ursprung, porträtt och landskap).
- Glöm inte att ta reda på formatet som bilderna bör vara i.
- Använd gärna pixlr-editor eller Photopea för beskärning. Squoosh för optimering.
- Justera i CSS-filen så att designen blir responsiv.
- Kom ihåg att maxbredden ej bör överstiga 1200px.

Ändra i  HTML-filen så bilden med solnedgång används istället och justera ditt grid så att den liknar de tre bilderna under.
